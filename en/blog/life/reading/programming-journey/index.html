My Journey in Programming

Four years ago, I was an ordinary liberal arts graduate. Today, I'm a full-stack engineer.

Starting Point: Zero Foundation

In 2019, I decided to switch to programming.

Phase 1 (3 months):

  • HTML/CSS/JavaScript basics
  • Study 4-6 hours daily
  • Built 20+ small projects

Phase 2 (6 months):

  • React ecosystem
  • Node.js backend development
  • Open source contributions

Growth: Accumulating Experience

Joined a startup, built products from scratch:

  • Led frontend architecture independently
  • Designed database structures
  • Full deployment and DevOps process

Transformation: Deep Technical Growth

Since 2022, deep diving into:

  • System design and architecture
  • Performance optimization
  • Team collaboration and project management

Advice for Beginners

  1. Hands-on practice - Project-driven learning
  2. Continuous learning - Tech evolves fast
  3. Build connections - Tech community matters
  4. Stay patient - Growth takes time

Future Outlook

Continue to deepen expertise in AI application development and cloud-native technologies.
